This contains patterns for two parts (LED-FLEX-R2 and LED-FLEX-R2-HANDLE), as
Gerber files.  The .MSK files contains holes that should be cut out, as
though for a solder paste stencil.  The .MSO files contain the part outline
(cut line) for each part.  

Material: Cut from 2 mil stainless sheet.
